### 1. Prerequisites

### 2. Setup
1. Clone this repository (or copy the files).
2. Run `npm install` to install dependencies.
3. Follow the instructions in **[FIREBASE_SETUP.md](./FIREBASE_SETUP.md)** to configure your database.
4. Create a `.env.local` file with your Firebase credentials (template in FIREBASE_SETUP.md).
